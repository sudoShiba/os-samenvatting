# Besturingssystemen Course Notes

Welkom bij de uitgebreide theorie-samenvattingen voor het vak Besturingssystemen (B-KUL-G0Q35A). Deze notities vatten de slides en colleges samen, en zijn opgebouwd rond de drie pilaren van het OSTEP-handboek (Operating Systems: Three Easy Pieces).

De leerstof is opgesplitst in de volgende kernconcepten:

1. **[Virtualization (Virtualisatie)](course_notes/virtualization.md)**
   Hoe het besturingssysteem de CPU en het geheugen virtualiseert zodat het lijkt alsof elk programma de hele computer voor zichzelf heeft.
2. **[Concurrency (Gelijktijdigheid)](course_notes/concurrency.md)**
   Hoe we omgaan met meerdere processen en threads die tegelijkertijd draaien en data delen.
3. **[Persistence (Persistentie)](course_notes/persistence.md)**
   Hoe we informatie permanent en veilig opslaan op disks met behulp van bestandssystemen en logging.
